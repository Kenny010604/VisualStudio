Elementos de Formato HTML

Los elementos de formato fueron diseñados para mostrar tipos especiales de texto:

  • <b> - Texto en negrita
  • <strong> - Texto importante
  • <i> - Texto en cursiva
  • <em> - Texto enfatizado
  • <mark> - Texto resaltado
  • <small> - Texto más pequeño
  • <del> - Texto eliminado
  • <ins> - Texto insertado
  • <sub> - Texto en subíndice
  • <sup> - Texto en superíndice

Elementos HTML <b> y <strong>

El elemento HTML <b> define texto en negrita, sin ninguna importancia adicional.

Ejemplo

Este texto está en negrita

El elemento HTML <strong> define texto con gran importancia. El contenido dentro se muestra generalmente en negrita.
