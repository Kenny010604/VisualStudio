Elementos HTML

Un elemento HTML está definido por una etiqueta de inicio, algún contenido y una etiqueta de cierre.

Elementos HTML Anidados

Los elementos HTML pueden estar anidados (esto significa que los elementos pueden contener otros elementos).

Todos los documentos HTML consisten en elementos HTML anidados.

El siguiente ejemplo contiene cuatro elementos HTML (<html>, <body>, <h1> y <p>):

<html>
    <body>
        <h1>Mi Primer Encabezado</h1>
        <p>Mi primer párrafo.</p>
    </body>
</html>
